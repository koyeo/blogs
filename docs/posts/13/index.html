解决 filecoin-ffi build 时 rust `edition2021` is required 错误

四月 1, 2022 · 1 分钟 · Koyeo

操作步骤#

git clone https://github.com/filecoin-project/filecoin-ffi.git
cd filecoin-ffi
./build.sh

解决思路#

  1. 尝试通过升级 rust 解决： rustup default nightly && rustup update — 失败
  2. 通过查看 filecoin-ffi 发布的最新版本，切换到该版本下编译，而非在 master 分支编译。 — 成功

注意： 编译时注意参考官网安装相关依赖：https://filwiki.cn/get-started/lotus/installation/#macos
